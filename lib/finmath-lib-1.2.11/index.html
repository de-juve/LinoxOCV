Contents of this Archive

The archive folder contains "finmath lib" as a packaged jar. The material is released under the license contained in LICENSE.txt. See also the file NOTICE.txt. In addition to the library (packed as a jar file), you will find an

  • API Documentation, documenting the use of each class,
and a set of
  • Spreadsheets, demoing the use of the library as a spreadsheet plugin.

Note: The spreadsheets require the installation of Obba, a middleware which allows to instantiate and use Java objects from within a spreadsheet.
